# RSA - Sistema Criptográfico Assimétrico 

Este projeto consiste na implementação do algoritmo **RSA**, incluindo a geração das chaves, criptografia e descriptografia de mensagens sem a utilização de bibliotecas criptograficas prontas.

O objetivo é demonstrar, de forma prática, o funcionamento de um dos algoritmos de criptografia assimétrica mais importantes da computação moderna.

Tal projeto está sendo produzido para compor e incrementar parte dos ensinamentos vistos na cadeira de Fundamentos da Matemática para Ciência da Computação 2 da UFCG.
